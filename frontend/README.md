# Orderflow frontend

[Create React App](https://github.com/facebook/create-react-app)으로 생성한 Frontend application입니다.

- node version: `v23.2.0`

## Getting started
### Requirements
```sh
# requires node 23+
brew install node
```

### 실행
```sh
# 3000 포트 중 /api/** 경로는 API 연결을 위한 8080 포트로 포워딩됩니다.
npm install
npm run start
```

### 빌드
```sh
npm install
npm run build
```
